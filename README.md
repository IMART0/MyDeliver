# Запросы для работы с api (пользователь)
## Страница профиля
1) Получить информацию о пользователе
```
GET /api/user/{id}
```
2) Изменить информацию о пользователе
```
PUT /api/user/{id}
```

## Страница создания заказа
1) Создать заказ
```
POST /api/order
```

2) Получить список всех заказов
```
GET /api/order
```
